Root 安全配置

修改 root 用户名

将 /etc/passwd 与 /etc/shadow 第一列的 root 改为想要的管理员名称

禁用 root 密码登录

修改 /etc/ssh/sshd_config ：
PermitRootLogin without-password

配置 ssh 密钥登录

方法一：服务器端.ssh/authorized_keys 文件添加客户端的公钥，配置文件权限为644
方法二：客户端指定使用服务器端的私钥登录,务必保存好私钥

禁用其它用户 sudo 切换到 root
